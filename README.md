## Overview
The Galamsey Reporting Platform is a web application that enables citizens to report illegal mining (galamsey) activities in a transparent, secure, and incentive-driven way.  
By combining **blockchain technology** and **AI**, the platform ensures data integrity, user trust, and rewards community participation.

---

## Problem
Illegal mining (galamsey) causes severe environmental damage, water pollution, and loss of livelihoods.  
Traditional reporting systems lack:
- Transparency
- Trust in data handling
- Motivation for citizens to report incidents

---

## Solution
This project provides:
- A **digital reporting platform** for galamsey activities
- **On-chain verification and rewards** using smart contracts
- **AI-powered summarization** of reports for faster insights

## On-Chain Functionality
- Each report submission triggers a smart contract function
- The contract records the report and rewards the user
- Users earn points/tokens visible in their connected wallet
- All transactions are verifiable on the Scroll blockchain

---

## User Flow
1. User signs in using OTP or Magic Link  
2. User connects wallet (MetaMask)  
3. User submits a galamsey report  
4. Smart contract processes the submission  
5. User earns a reward on-chain  
6. AI summarizes the report for analysis
